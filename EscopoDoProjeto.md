
# Escopo do Projeto: Link do Cidadão (Rio Claro - RJ)

Este documento define as diretrizes, funcionalidades e o roteiro de desenvolvimento do totem interativo **Link do Cidadão**, projetado para a Prefeitura Municipal de Rio Claro - RJ.

## 1. Objetivo do Projeto
Transformar a interação entre o cidadão e a gestão pública através de um ponto de autoatendimento inteligente, inclusivo e multilíngue, reduzindo filas e democratizando o acesso à informação oficial.

## 2. Funcionalidades Atuais (V1.2.1 - Demo Ready)
- **Interface Multilíngue**: Suporte completo para Português (Brasil), Inglês e Espanhol.
- **Acessibilidade Inclusiva**:
    - **Modo Cadeirante**: Reposicionamento de elementos para fácil alcance.
    - **Alto Contraste**: Otimizado para cidadãos com baixa visão.
    - **Voz da IA (TTS)**: O assistente municipal fala as respostas, garantindo acesso para analfabetos ou deficientes visuais.
- **Assistente AI Rio Claro**: Interface de chat em tempo real integrada com Google Gemini para dúvidas sobre serviços municipais.
- **Portal de Serviços**: Atalhos rápidos para IPTU, Certidões, Ouvidoria, Agendamento de Saúde, Transparência e Site Oficial.
- **Dinamismo em Tempo Real**: 
    - **Widget de Clima**: Temperatura atual de Rio Claro.
    - **Plantão Rio Claro**: Ticker de notícias dinâmico com atualizações constantes.
- **Gestão de Emergência**: Banner crítico da Defesa Civil gerenciado via Admin.
- **Auditabilidade**: Registro de operações e logs via repositório GitHub oficial.

## 3. Público-Alvo
- Cidadãos de Rio Claro - RJ.
- Turistas e visitantes nacionais e internacionais.
- Pessoas com mobilidade reduzida ou deficiências.

## 4. Pilares Tecnológicos
- **Frontend**: Next.js 15, Tailwind CSS, ShadCN UI.
- **Inteligência Artificial**: Google Genkit (Gemini 2.5 Flash), Gemini TTS.
- **Transparência**: Gestão via GitHub Auditável.
